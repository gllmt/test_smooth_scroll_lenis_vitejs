## Installation

Use the package manager of your choice.

```bash
npm install
```
or
```bash
yarn install
```

## Usage

Launch the development server :

```bash
npm run dev
```
or
```bash
yarn dev
```

## Files

Write your html in /index.html

Write your javascript in /main.js

Manage your assets in /assets folder (sass, css, images, fonts...)
